Exercice Git & GitHub

Qu'est-ce que Git ?

Git est un logiciel qui permet le versionning des projets. Il permet aussi un système de branches pour le travail collaboratif

Il permet de garder un historique des modifications de fichier sur sa machine. Il permet également de déposer les fichiers sur un serveur distant


Et GitHub alors ?

GitHub est un service de stockage des fichiers en ligne. Il permet de conserver le travail et y acceder depuis n'importe quelle machine executant Git
